# Shift Watch

## Overview

Shift Watchis a full-stack web application designed to collect, analyse, and report hospital staffing data over time. The platform provides objective evidence of staffing levels, patient acuity, and operational workload to support workforce planning, quality improvement initiatives, and advocacy.

The application enables healthcare staff to record staffing information for each shift, review historical data, generate charts and reports, and export data for further analysis.

## V1 Features

* Record staffing levels for each shift
* Record planned vs actual staffing
* Record nursing skill mix (RN, EN, AIN, NUM, etc.)
* Record permanent, casual, and agency staff
* Record patient numbers, high acuity, and high dependency patients
* Record operational metrics (e.g. falls, clinical reviews, rapid responses, Code Blue)
* Search and filter historical shift data
* Generate charts and dashboards
* Export reports as CSV

## Technology Stack

* **Frontend:** React
* **Backend:** Node.js + Express
* **Database:** Microsoft SQL Server
* **Authentication:** JWT (planned)
* **Charts:** Recharts
* **Version Control:** Git & GitHub

## Project Goal

To build a secure, data-driven platform that provides reliable evidence of staffing levels, patient acuity, and workforce pressures. The collected data will support trend analysis, reporting, and informed discussions with healthcare organisations, unions, and policymakers.
